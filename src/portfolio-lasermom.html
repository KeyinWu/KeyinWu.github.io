Laser Mom is an explorational hide-and-seek game combining both physical props and virtual worlds. The guest will play a kid who has to hide behind the physical table from a virtual mom. With Passthrough in Oculus, it's utilizing augmented reality world. This game is a two-week project in Building Virtual Worlds course and specially designed for naive guests with few instructions.

Contribution

Gameplay Programming (Game Flow, NPC AI, State Machine, Game Manager, Event Manager...), Calibration Tool, Level Design, Playtesting
